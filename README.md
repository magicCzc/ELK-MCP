## 简介
elk-MCP 是日志查询与工作流集成的后端服务，支持基于 Elasticsearch 的多索引查询、字段精简、分页与国际化预留。

## 快速开始
- 配置 `backend/.env`（参考 `backend/.env.example`）
- 启动后端服务并使用 `/api/logs/query` 进行查询
